# Fluxo de Análise

## Etapas principais

1. **SQL**
   - Extração de dados das tabelas relevantes da base WideWorldImporters.
   - Criação de consultas para consolidar informações sobre pedidos.

2. **Python**
   - Limpeza e tratamento de valores irregulares.
   - Análise preliminar dos dados.
   - Exportação de uma base consolidada para o Power BI.

3. **Power BI**
   - Criação de dashboards interativos para análise da performance de pedidos.
   - Construção de visualizações e KPIs para destacar insights importantes.
